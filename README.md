**СУТЬ ПРИЛОЖЕНИЯ**

--------------------------

Пользователь сможет создавать коллекции игр, оставлять отзывы и оценки о играх всех платформ (PC (Steam, Epic Games, Origin), PS4/5, XBOX и т.д.). Также, пользователь сможет просмотреть статистику по играм.
